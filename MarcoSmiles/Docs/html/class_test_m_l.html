◆ ReteNeurale()

static int TestML.ReteNeurale ( float[]  features)
static

Predice la nota associata alle features passato come parametro.

Usa le matrici W1, B1, W2 e B2 per effttuare i calcoli. Restituisce un indice che va da 0 a 23, corrispondente alla nota da suonare. La funzione supporta il caso in cui il dataset contenga meno note rispetto agli indici 0-23. Questo è fatto calcolando in fase di computazione la grandezza delle matrici W1, B1, W2, B2. Conseguentemente, la funzione ReteNeurale, restituirà indici che si trovano fra il range di note allenate.
